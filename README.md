# Hitachi Graphite

Сервис для сбора метрик производительности и использования ресурсов СХД Hitachi с последующей отправкой данных в Graphite.

Проект был разработан для использования во внутренней системе мониторинга инфраструктуры.

## Возможности

- сбор метрик производительности и ёмкости СХД Hitachi
- получение данных через Hitachi Tuning Manager API
- отправка метрик в Graphite
- поддержка нескольких СХД одновременно
- параллельный сбор данных с использованием пула воркеров
- настройка конфигурации через YAML
- настройка списка собираемых метрик через отдельный файл
- логирование работы приложения

## Стек

- Go
- Hitachi Tuning Manager API
- Graphite

## Конфигурация

Перед запуском необходимо создать файл `config.yml`.

В качестве шаблона можно использовать `configExample.yml`.

Настраиваются:

- подключение к Graphite
- параметры доступа к Hitachi Tuning Manager API
- список СХД
- количество воркеров
- параметры логирования

### Собираемые метрики

Список собираемых метрик и параметры их обработки задаются в файле `metrics.yml`.

## Структура проекта

```
.
├── config          # загрузка конфигурации
├── getData         # получение данных из Hitachi API
├── sendData        # формирование и отправка метрик в Graphite
├── utils           # вспомогательные функции
├── configExample.yml
├── metrics.yml
└── main.go
```
